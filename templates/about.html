About Park Smart UK

Park Smart UK is an interactive, web-based mapping system that displays street-specific parking availability and restrictions across the United Kingdom. It combines geospatial mapping, computer vision OCR, and machine-learning availability forecasting to reduce parking stress and help drivers avoid costly Penalty Charge Notices (PCNs).

Key Features

  • Interactive Leaflet.js Map — colour-coded parking zones with real-time filter controls.
  • OpenCV OCR Engine — upload a photo of any UK parking sign; the system extracts restriction details automatically.
  • ML Availability Predictor — TensorFlow LSTM model forecasts occupancy trends based on historical data.
  • Crowdsourced Data — any user can contribute new signs, expanding map coverage organically.
  • Disabled Bay Finder — dedicated filter to locate Blue Badge bays instantly.

Technology Stack

Backend Python 3.11, Flask 3.x
Computer VisionOpenCV 4.x, pytesseract (Tesseract 5)
Machine LearningTensorFlow 2.x / Keras LSTM; scikit-learn GBR fallback
Mapping Leaflet.js 1.9 + OpenStreetMap tiles (100% free, no API key)
Data JSON store (dev); PostgreSQL + PostGIS (production)
Dev EnvironmentAnaconda, VS Code

Intellectual Property

The project uses only open-source software (Python, TensorFlow, OpenCV, Flask, Leaflet.js). All mapping data is sourced from OpenStreetMap (ODbL licence) or original datasets. No third-party proprietary APIs are used.

Project Aims

  • Reduce urban congestion caused by drivers searching for parking.
  • Help drivers comply with local council regulations and avoid PCNs.
  • Improve accessibility by surfacing disabled and free parking prominently.
  • Demonstrate the value of computer vision applied to civic problems.
